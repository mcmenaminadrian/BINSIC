Binsic Is Not Sinclair Instruction Code

BINSIC is an attempt to implement Sinclair's ZX80/ZX81 BASIC in Groovy.
(Mainly as an interpreter but with some DSL-like elements too).

(With a few added bits and pieces that were missing Back In The Day too, I expect).
